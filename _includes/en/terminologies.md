## Terminology

I use abreviates in the pattern here are them to help you read the pattern

* Rnd : Round
* ch : Chain 
* sc : Single crochet
* ss : Slip stich
* BLO : Back loop only
* FLO : Front loop only
* inc : Increase
* dec : Decrease
* dc : Double crochet

For each round you'll find the number of the round, the instructions and the total number of stitches between parenthesis
